Lists

This is how you make an unordered list:

  • Add a "ul" element (it stands for unordered list)
  • Add each item in its own "li" element
  • They don't need to be in any particular order

and that "li" elements allows nested paragraphs?

Because the Mozilla Developer Network (MDN) says so. whenever you're not sure
about a particular element, do a quick Google search for "MDN 'some-element'".
